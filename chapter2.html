2.5 - Absolute Value Equations

Notes

  • The absolute value or distance can never be negative as absolute value is the distance between 2 numbers.
  • If you graph an absolute-value function, it should resemble a V-shape.
  • To write an equation, it would look like |x-midpoint|=distance between 2 numbers. It would be minus if the midpoint is positive and plus if the midpoint is negative.
  • Make 2 cases.
  • When the coefficient of the "x" value is not 1 you have to set the absolute value to 0. If the coefficient is 1, the opposite of the number in place of the midpoint will be your midpoint.
  • For example: |x-5|=10, 5 is your midpoint and 10 is your distance.
  • For example: |2x-6|=10, 3 is your midpoint and 5 is your distance.
  • Isolate the absolute value and then make the second case where you make the other side negative.

2.8 - Literal Equations

Notes

  • Basically you are isolating to find the value of only one variable.
  • If u have any equation like u=kx+yx and it is asking for you to find the value of x you would distribute the right side.
  • u=x(k+y)
  • u/(k+y) = x
