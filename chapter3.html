Chapter 3 - Linear Functions

3.1 - Linear Equations

Notes

  • Standard Form: Ax+By=C
  • A can never be negative.
  • A, B are variables and they can never be 0 at the same time!
  • C is the constant.
  • A, B and C are all integers. THEY CAN’T BE FRACTIONS OR DECIMALS.
  • GCF of A, B, and C = 1
  • To graph a line, use the x and y-intercepts.
  • Linear Function is a function whose graph has a straight line.

3.2 - Solving Linear Equations

Notes

  • When the question asks you to solve by finding the 0 you are basically finding the x-intercept.
  • Always set the “y” to 0 and then solve.
  • When finding the zero remember to distribute to the RIGHT, and write f(x) on the left.

3.3 - Rate of Change & Slope

Notes

  • 4 types of slope: Positive Slope, Negative Slope, Zero Slope, and Undefined Slope.
  • Positive Slope is from bottom left to top right.
  • Negative Slope is from top left to bottom right.
  • Zero Slope is always a horizontal line. You know its a zero slope if 0 is in the numerator ( eg. 0/5 ) because 0/5 = 0.
  • Undefined Slope is always a vertical line. You know its an undefined slope if 0 is in the denominator ( eg. 5/0 ) because 5/0 is mathematically incorrect.
  • You can find the slope by 2 points, make a table and a graph.

3.4 - Direct Variation

Notes

  • y=kx
  • k is the constant of variation. k never equals 0.
  • A direct variation passes through the origin.
  • The domain and range have proportional relationships.
  • d=rt (d (y) = distance, r (m or k) = rate, and t(x) = time)

Notes

3.6 - Proportional or Non-Proportional

Notes

  • An equation can be non-proportional if it doesn’t pass through the origin.
  • A proportional function will always be linear. It will always have a constant "k" value.
