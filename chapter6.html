Chapter 6 - Systems of Linear Equations and Inequalities

6.1 - Graphing Systems of Equations

Notes

  • Vocabulary Terms: System of Equation: two equations in a given set
    Consistent: if a system has at least one solution
    Independent: if a system has exactly one solution
    Dependent: if a system has infinite solutions
    Inconsistent: If a system has no solution

  • Intersecting lines
    Exactly 1 solution
    Consistent
    Independent

  • Same line
    Infinite solutions
    Consistent
    Dependent

  • Parallel Lines
    No solution
    Inconsistent

6.2 - Substitution

Notes

  • Substitution is when you find a variable like a literal equation, Substitute it for the other variable in the other equation, and then solve. Then, plug that variable into one of the equations and solve. Now you have both variables.
  • Steps to solve using Substitution:
    Step 1: When necessary, solve at least one equation for one variable.
    Step 2: Substitute the resulting expression from step 1 into the other equation to replace the variable. Then solve the equation.
    Step 3: Substitute the value from Step 2 into either equation and solve for the variable. Write the solution as an ordered pair.
    Step 4: You can check by plugging the values of x and y into either equation.

6.3 - Elimination using Addition & Subtraction

Notes for 6.3 & 6.4

  • When using elimination, you must cancel out one variable to solve for the other.
  • To cancel out the variable, you have to manipulate the equations to make them cancel out.
  • Use elimination using addition if you see opposite coefficients. (+-)
  • Use elimination using subtraction if you same coefficients.
  • If you get variables like m and n, you would write m as "x" and n as "y". (Remember alphabetical order)
  • When multiplying a variable find the LCM of both coefficients.

6.6 - Systems of Inequalities

  • A ≤ or ≥ symbol means solid line.
  • A greater than or lesser than symbol means dotted line.
  • The solution is the overlapping regions.
  • You shade above the line if it's > or ≥ and you shade below the line if it's lesser than or ≤.
  • For values like time, distance, and speed in word problems where a negative value is not possible, put a closed circle on the x and y axes (if it crosses both on the graph), since the value technically cannot go below zero. This is because it represents an inequality.
  • Even though parallel lines in systems of equations mean no solution, they can have a solution in an inequality if there is a region where both inequalities are shaded.
