Chapter 1 - Expressions, Equations, & Functions

1.1 - Variables and Expressions

Notes

  • Variables are unknown values or numbers.
  • Expressions don’t have “equal to” signs. Equations do.
  • Example: 3x + 4y=18. 3 and 4 are the coefficients. 3x, 4y, 18 are terms. 18 is a constant. This is an equation.

1.4 - Distributive Property

Notes

  • Make sure to draw parenthesis when you are using distributive property.
  • If you are trying to group the variables, take out the coefficient (if possible).
  • Example: ab + ac = a(b + c)

1.5 - Equations

Notes

  • In equations which are open sentences, you have to substitute the variable with each number in the set.
  • A replacement set is a set of numbers that replaces the variable and forms a true statement.
  • Write your answer in {}
  • In equations with 2 variables, you replace the second variable to find answer (easier by making table).
