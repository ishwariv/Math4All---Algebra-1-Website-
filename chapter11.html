Chapter 11 - Radical Functions & Equations

11.2 - Rational Functions

Notes

  • Parent Function: f(x) = 1/x
  • Type of Graph: hyperbola
  • domain: {x|x ≠ 0}
  • range: {y|y ≠ 0}
  • Excluded Values: Any value that makes your denomiator 0 is considered excluded values.
  • vertical asymptote: x=b
  • horizontal asymptote: y=c
  • ALWAYS graph the asymptotes with dotted lines.

11.3 - Simplifying Rational Functions

Notes

  • While simplifying:
    - Factor numerator
    - Factor denominator
    - Simplify the expression
    - State the excluded values
  • While finding the zeros:
    - Set the numerator to 0
    The reason behind this is that 0 over anything will always result in 0. So it doesnt matter what the denominator is. Thus, when asked to find the zero always set the numerator to 0.
    - Factor
    - Simplify
    - State the zeros
